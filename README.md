# Consejito del día
Una Single Page Application (SPA) sencilla y responsiva que muestra consejitos inspiradores de manera aleatoria usando una API externa. 

# Descripción
Al hacer clic en el botón, la app muestra un nuevo "consejito del día" obtenido en tiempo real desde una API.

Proyecto 3 del bootcamp Femcoders, en la que trabajamos con:
- HTML
- CSS
- Javascript
- Figma
- Visual Studio Code
- Trello
- Git / GitHub

# Funcionalidades
- Obtiene consejos aleatorios desde la API. 
- Diseño adaptable a móviles y pantallas grandes.
- Logo a la izquierda y título centrado. 
- Código limpio, semántico y simple.
